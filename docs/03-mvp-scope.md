# MVP Scope

## MVP Objective

Memvalidasi apakah kombinasi budget tracking dan weekly financial check-in dapat membantu user lebih disiplin dalam mengelola pengeluaran.

---

## Features Included

### Authentication

- Register
- Login
- Logout

### Onboarding

- Set Current Balance
- Set Monthly Budget
- Set Reminder Preference

### Dashboard

- Current Balance
- Monthly Budget
- Used Budget
- Remaining Budget
- Financial Status

### Transaction Management

- Add Income
- Add Expense
- Edit Transaction
- Delete Transaction
- Transaction History

### Budget Management

- Create Budget
- Update Budget
- Budget Progress Tracking

### Weekly Financial Check-In

- Friday Reminder
- Budget Evaluation
- Spending Summary
- Financial Status
- Recommendation

---

## Out of Scope

Fitur berikut tidak termasuk dalam MVP:

- AI Financial Assistant
- Savings Goals
- Investment Tracking
- Debt Management
- Gamification
- Achievement System
