# Mindspend V2

## Overview

Mindspend adalah aplikasi personal finance yang membantu user mengelola budget bulanan dengan lebih sadar melalui expense tracking, budgeting, dan weekly financial check-in.

Fokus utama produk bukan hanya mencatat transaksi, tetapi membantu user membangun financial awareness dan mengontrol spending behavior sebelum terjadi overspending.

---

## Product Vision

Membantu user membangun kebiasaan finansial yang lebih sehat melalui budgeting sederhana, reminder yang relevan, dan insight yang actionable.
